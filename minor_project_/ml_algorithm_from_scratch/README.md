# ML Algorithms From Scratch

This repository is my hands-on implementation of a few core machine learning algorithms in pure Python.  
The goal was not to copy a library call, but to understand how these models actually work under the hood.

## What is inside

- **KNN Classifier**
- **KNN Regressor**
- **Linear Regression**
- **Logistic Regression**

## Why I made this

I wanted to move beyond just using `sklearn` and understand the basic ideas behind these algorithms:
distance calculation, prediction logic, gradient descent, and decision boundaries.

A project like this helps a lot with the fundamentals. It also makes debugging much easier later, because you know what the model is doing instead of treating it like a black box.

## What I learned

- How KNN uses nearest neighbors for classification and regression
- How Linear Regression fits a line to data
- How Logistic Regression works for binary classification
- Why preprocessing and feature scaling matter
- How much logic is hidden inside high-level ML libraries
